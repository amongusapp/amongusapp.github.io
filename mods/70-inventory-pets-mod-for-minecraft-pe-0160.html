Inventory Pets mod for Minecraft PE 0.16.0

Mods / 0.16.2
50 234
3.10.16
+28
This mod for MCPE adds talismans that will apply different magic effects on your character. There will be 31 talisman in total, you can craft as many of them as you want, but active will be only the ones you placed in hotbar. To make them work, you also need to feed them from time to time - each of them needs different food (description of the required food and the abilities of pets are in the spoiler). They can make you more adapted for survival than ever before.

How to install mod:
  1. Download and install BlockLauncher PRO
  2. Download and unzip archive to your device (you must get .js or .modpkg file)
  3. Open BlockLauncher, go to Manage ModPE scripts - Import - Local Storage
  4. Go to Download folder and select mod that you want to install
  5. If mod has a texture pack, you must install it before script
