Actual Guns is one of a few realistic weapon addons for Minecraft PE in existence. All guns are fully functional and working in the same way as it's real-life prototypes. The only two unrealistic features - automatic reloading process and shooting without using bullets are made for the player's convenience. Another thing worth mentioning - guns cannot be crafted in survival, they can be acquired only via command.

Credits: AzozGamer936

How to get guns


As I've said earlier - the only way to get guns is through commands. Type in the following command to give yourself a particular weapon, in this case, grenade - /give @p gun:grenade. To get all of the guns the addon offers, type in chat - /function getallguns.


The minigun - is a fun weapon, but it's heavy, and you're not Arnold Schwarzenegger, so you're going to be moving much slower while using it.




With a double-barrel gun, you can fantasize you're playing Doom, especially in the Nether.


Smoke Grenades are special, have you ever dreamed that we could have a 3D smoke in Minecraft?

How to install addons:
  1. Download addon straight from your device
  2. Find and open .mcpack or .mcworld files that you download earlier
  3. Click it to import into Minecraft Pocket Edition
  4. Open the game
  5. Create a new world or edit an existing world
  6. Scroll down to Add-ons section and select the ones you want to apply
